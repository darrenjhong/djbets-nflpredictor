# 🏈 DJBets NFL Predictor v9.5-S

NFL win-probability predictor using XGBoost + Streamlit.
- Pre-trained model (no waiting)
- Team logos and spreads
- Offline-ready, Streamlit-compatible

## 🚀 Run Locally
pip install -r requirements.txt
streamlit run streamlit_app.py

## 🌐 Deploy to Streamlit Cloud
1. Push to GitHub
2. Go to https://share.streamlit.io
3. Set entrypoint to `streamlit_app.py`
4. Done!
